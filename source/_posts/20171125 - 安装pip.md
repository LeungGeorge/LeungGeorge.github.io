# 安装pip
## 下载get-pip.py

```
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
```

<!--more-->

## 运行

```
python get-pip.py
```

## 升级pip

```
pip install -U pip
```



> 引用  
> https://pip.pypa.io/en/latest/installing/#id7
